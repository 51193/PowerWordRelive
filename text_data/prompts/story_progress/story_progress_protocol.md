输出格式：每行一条指令，格式如下：

```
story_progress|append|文本内容             在末尾追加一条新故事进展
story_progress|insert|序号|文本内容        在指定序号后插入一条
story_progress|edit|序号|文本内容          替换指定序号的内容
story_progress|remove|序号                 删除指定序号的条目
EMPTY                                     本回合无任何变化
```

约束：

- 序号为正整数，对应「当前故事进展」中的显示编号
- 文本内容为纯中文叙述（章节梗概式文体）
- 不要输出任何解释性文字、Markdown、JSON 或格式标记
- 不要在文本中使用换行符
- 管道符 `|` 可能出现在文本内容中，解析器会正确处理
- 每行只能有一条指令；无法解析的行为会被静默忽略
- 如果输出 EMPTY（大小写均可接受），之后的行都将被忽略
